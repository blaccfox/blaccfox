
	為る (to do)

		Miscellaneous 
		
		彼ら　を　為よ　。

		彼ら　を　為ろ　。

		Hypothetical Conjugation Plain

		彼　を　すれば、　そして、　いい　です　。

		彼女　を　すれば　し　無い、　そして、　いい　じゃない　です　。

		暑　く　すれば、　そして、　いい　です　。

		寒　く　すれば　し　無い、　そして、　いい　じゃない　です　。

		Hypothetical Conjugation Polite

		彼　を　すます　れば・ば・なる、　いい　です　。

		彼女　を　すません　れば・ば・なる、　いい　じゃない　です　。

		暑　く　する　れば・ば・なる、　そして、　いい　です　。

		寒　く　する　無ければ　れば・ば・なる、　そして、　いい　じゃない　です　。





		彼　を　為れば、　そして、　いい　です　。

		彼女　を　為れば　し　無い、　そして、　いい　じゃない　です　。

		彼　を　為ます　れば、　いい　です　。

		//暑　く　ければ、　そして、　いい　です　。

		//寒　く　無ければ、　そして、　いい　じゃない　です　。

		暑　く　為る　れば・ば・なら、　そして、　いい　です　。

		寒　く　為る　無い　れば・ば・なら、　そして、　いい　じゃない　です　。

		彼ら　は　為る　れば・ば・なら、　そして　いい　です　。

		彼ら　は　為る　無い　れば・ば・なら、　そして　いい　じゃない　です　。
