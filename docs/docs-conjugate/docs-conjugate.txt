
	する (to do)

		Imperative Conjugation Plain
		
		彼ら　を　すよ　。

		Imperative Conjugation Polite

		彼ら　を　すろ　。

		Formal Conjugation

		彼　を　します　。

		彼女　を　しません　。

		Perfective Conjugation

		Conjunctive Conjugation 

		Hypothetical Conjugation Plain

		彼　を　すれば、　そして、　いい　です　。

		彼女　を　しない　すれば、　そして、　いい　じゃない　です　。

		暑　く　すれば、　そして、　いい　です　。

		寒　く　しない すれば、　そして、　いい　じゃない　です　。

		Hypothetical Conjugation Polite

		彼　を　します　れば・ば・なら、　いい　です　。

		彼女　を　しません　れば・ば・なら、　いい　じゃない　です　。

		暑　く　する・すれ　れば・ば・なら、　そして、　いい　です　。

		寒　く　する・すれ　無ければ　れば・ば・なら、　そして、　いい　じゃない　です　。
