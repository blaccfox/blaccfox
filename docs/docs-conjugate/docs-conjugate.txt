
	為る (to do)

		Miscellaneous 
		
		彼ら　を　為よ　。

		彼ら　を　為ろ　。

		Hypothetical Conjugation Plain

		彼　を　すれば、　そして、　いい　です　。

		彼女　を　すれば　し　無い、　そして、　いい　じゃない　です　。

		暑　く　すれば、　そして、　いい　です　。

		寒　く　すれば　し　無い、　そして、　いい　じゃない　です　。

		Hypothetical Conjugation Polite

		彼　を　すます　れば・ば・なる、　いい　です　。

		彼女　を　すません　れば・ば・なる、　いい　じゃない　です　。

		暑　く　する　れば・ば・なる、　そして、　いい　です　。

		寒　く　する　無ければ　れば・ば・なる、　そして、　いい　じゃない　です　。
